## Quick How-to for final assembly
1. Create screenshots of GUI pages by running:

```powershell
# in project root (Windows PowerShell)
streamlit run app.py
```

Then capture the relevant pages in the browser and save to `screenshots/`.

2. Export ER diagram to `ER_DIAGRAM.png` and add it to the repo.

3. Create `final_report.pdf` by combining markdown files and images (use pandoc or a Word processor):

```powershell
# Example (requires pandoc installed):
pandoc COVER_PAGE.md PROJECT_REPORT.md USER_REQUIREMENT_SPEC.md DELIVERABLES.md -o final_report.pdf --pdf-engine=xelatex
```

4. Add GitHub repo link to `README.md` and push.
